# Churn Prediction Project

## Descripción

Proyecto de machine learning orientado a la predicción de abandono de clientes (churn). Se utilizan técnicas de análisis de datos y modelos de clasificación en Python.

## Tecnologías

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Objetivo

Analizar el comportamiento de los clientes y desarrollar un modelo capaz de identificar aquellos con mayor probabilidad de abandonar el servicio.

## Ejecución

Clonar el repositorio:

```bash id="x1"
git clone https://github.com/tu-usuario/churn-project.git
```

Acceder al directorio:

```bash id="x2"
cd churn-project
```

Instalar dependencias:

```bash id="x3"
pip install -r requirements.txt
```

Ejecutar el proyecto:

```bash id="x4"
python main.py
```

## Resultados

* Modelo de clasificación para predicción de churn
* Evaluación mediante métricas como accuracy, precision y recall
* Enfoque en la detección de clientes con mayor riesgo de abandono
