# 📊 Churn Prediction Project

## 📌 Descripción

Proyecto de Machine Learning para predecir el abandono de clientes (*churn*) utilizando técnicas de análisis de datos y modelos predictivos en Python.

---

## 🚀 Tecnologías utilizadas

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

---

## 📊 Objetivo

El objetivo de este proyecto es analizar el comportamiento de los clientes y construir un modelo capaz de identificar aquellos con mayor probabilidad de abandonar el servicio.

---

## ⚙️ Cómo ejecutar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/tu-usuario/churn-project.git
```

2. Accede al directorio:

```bash
cd churn-project
```

3. Instala las dependencias:

```bash
pip install -r requirements.txt
```

4. Ejecuta el proyecto:

```bash
python main.py
```
